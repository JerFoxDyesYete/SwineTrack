SwineTrack - Right on Time, Healthy Swine
-
SwineTrack is an IoT-based swine health and pig pen monitoring system designed to help hog raisers detect early health risks in pigs. The system continuously monitors swine body temperature, ammonia levels, and humidity, and displays the data through a mobile application.
